# About TechPulse Daily

TechPulse Daily is an automated tech news aggregator that curates the most interesting stories from over 50 authoritative sources.

## How It Works

Our algorithm scans top tech sources every few hours and selects the best articles based on:

- **Community validation** - Posts with high engagement on HN, Lobsters
- **Source authority** - Original content from GitHub, Netflix, Cloudflare engineering blogs
- **Title quality** - Headlines that inform, not clickbait
- **Topic relevance** - AI, security, startups, programming
